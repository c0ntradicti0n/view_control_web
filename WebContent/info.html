How to use

  1. Look for a document, that fits your interests by opening a collection of documents from the tree tables on the right side.
  2. Read as you like
  3. If you find a piece of text explaining a difference, just select it with the mouse and click the annotate button on the right side (right at the top of the page).
  4. Coming in the future are a custom user notebook for your made annotations and an app for Andoid mobile phones will appear here.
  5. Watch out in the video for your notes:
  6. Clone and download the flight simulator to run and fly it yourselve. hal run it (on Ubuntu 18.04, install Java 8 and Maven before): 
    
                git clone https://github.com/c0ntradicti0n/hal.git
                cd hal
                mvn package
                java -jar target/hal-1-jar-with-dependencies.jar --web
    Use the arrows for rotating yourself and the 'a', 's', 'd', 'w' keys for flying.
